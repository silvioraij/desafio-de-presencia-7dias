# Desafío de Presencia – 7 días

## Descripción

Este mini juego de mindfulness ofrece un reto de siete días para cultivar la presencia y la calma interior. Cada día propone una micro‑práctica con un temporizador y una animación de respiración. Al final de cada sesión se muestra una reflexión y una frase inspiradora. El progreso del usuario se almacena en `localStorage` para que pueda continuar donde lo dejó. Al completar los siete días, el sitio genera un certificado PNG descargable con el nombre y la fecha.

## Paleta de colores

- **Ocean Deep Blue**: #5D768E  
- **Warm Sandy Beige**: #C8B39B  
- **Ivory Breeze**: #F8EFEB

## Tipografías

- Inter o Nunito para el texto.  
- Playfair Display para los títulos.

## Cómo correr localmente

1. Clona este repositorio.  
2. Abre `index.html` en tu navegador o sirve la carpeta con un servidor estático.  
3. Asegúrate de que los archivos `styles.css` y `script.js` se encuentren en el mismo directorio.

## Despliegue en GitHub Pages

Este repositorio está configurado para desplegarse desde la rama `main` en GitHub Pages. Cada vez que hagas `push` a `main`, GitHub generará y publicará el sitio automáticamente.

## Accesibilidad y UX

- Modo claro con contraste suficiente.  
- Navegación por teclado y foco visible.  
- Animaciones suaves y sin distracciones.  
- Progreso guardado en localStorage y botón de reinicio.
